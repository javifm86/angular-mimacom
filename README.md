# angular-challenge
Angular + TailwindCSS application.

## npm scripts
- `npm start`: Launch `ng serve` for development mode.
- `npm run build`: Launch `ng build`.
- `npm run build:prod`: Generate app for production with aot.

## run api server locally
    cd server-api
    npm install
    npm start
